# Alura Quiz - Attack on Titan
![GitHub]() ![GitHub Repo stars]()

Este projeto foi criado durante a Imersão React Next.js da Alura.
O projeto se trata de um Quiz com o tema de Attack on Titan (tema escolhido por mim), que permite desafiar as pessoas para ver quem realmente sabe sobre café. Eu tentei escolher perguntas que fossem difíceis, e ao mesmo tempo, curiosidades interessantes para quem for responder.

## Tecnologias Utilizadas
- next
- styled-components
- eslint
- prop-types
- framer-motion
